UML-Editor - приложение, предназначенное для создания и редактирования диаграмм нотации UML Activity.

Добавление элемента на рабочую область:
Чтобы добавить элемент на рабочую область, перетащите их Панели инструментов, расположенной в левой части приложения, путём зажатия ЛКМ.

Соединение элементов:
Чтобы соединить элементы, выделите два нужных для соединения объекта на рабочей области и нажмите Q.

Сгибание стрелки:
Чтобы согнуть стрелку, нажмите ЛКМ по стрелке в нужном месте, чтобы добавить точку изгиба. Перетащите точку изгиба с помощью зажатия ЛКМ, чтобы подобрать нужный вам угол.

Редактирование элемента:
Чтобы изменить элемент, нажмите ЛКМ по нему, и справа появится панель редактирования, где вы сможете изменить нужные Вам параметры.

Удаление элемента:
Чтобы удалить элемент, нажмите по нему ЛКМ и в панели редактирования нажмите на кнопку "Удалить", либо выделите элемент и нажмите Delete.

Дублирование элемента:
Чтобы дублировать элемент, нажмите по нему ЛКМ и в панели редактирования нажмите на кнопку "Дублировать".

Сохранение:
Чтобы сохранить диаграмму, нажмите на раздел верхнего меню "Файл" и выберите "Сохранить" для быстрого сохранения с временной меткой в папку saves в папке с исполняемым файлом приложения, либо нажмите "Сохранить как" для сохранения с выбранным названием в конкретную папку.

Открытие:
Чтобы открыть ранее сохранённую диаграмму, нажмите на раздел верхнего меню "Открыть" и выберите файл с расширением .chep.

Вставка изображения:
Чтобы вставить изображение, нажмите на раздел верхнего меню "Вставка" и выберите "Изображение". Выберите изображение, размер которого не превышает 200х200.

Просмотр статистики:
Чтобы посмотреть статистику, нажмите на раздел верхнего меню "Статистика" и выберите "Запустить статистику".

Горячие клавиши:
Создание элементов:
Decision - '1'
Start event - '2'
End event - '3'
Splitter - '4'
Merge - '5'
Active state - '6'
Sending signal - '7'
Signal receipt - '8'
Transition - '9'

Прочее:
Соединение выбранных элементов - Q
Удаление выбранного элемента - Delete
Выделить все элементы на диаграмме - Ctrl+A
Дублировать выделенные элементы - Ctrl+D
Выделение - зажатый Ctrl и нажатие ЛКМ по элементу
